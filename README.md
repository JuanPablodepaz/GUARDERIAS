# 🚀 Guarderías - Proyecto Solana (Solana Playground)

Este proyecto fue desarrollado utilizando **Solana Playground (web)**, una plataforma online para crear, compilar y probar programas en Solana sin necesidad de instalación local.

---

## 🌐 Plataforma utilizada

👉 https://beta.solpg.io/

---

## 📁 Estructura del Proyecto
guarderia1/
│
├── client/
│ └── client.ts # Cliente para interactuar con el programa
│
├── src/
│ └── lib.rs # Programa en Rust (Smart Contract)
│
├── tests/
│ └── anchor.test.ts # Pruebas del programa

---

## ⚙️ Tecnologías Utilizadas

- 🟣 Solana Blockchain  
- ⚓ Anchor Framework (Playground)  
- 🦀 Rust  
- 🟨 TypeScript  

---

## 📌 Funcionalidad

El proyecto permite:

- ✔ Crear registros en blockchain  
- ✔ Leer información  
- ✔ Actualizar datos  
- ✔ Eliminar registros  

---

## 🚀 Cómo usar el proyecto

1. Abrir Solana Playground  
2. Crear un nuevo proyecto  
3. Copiar los archivos de este repositorio  
4. Pegarlos en el entorno web  
5. Compilar y ejecutar desde la plataforma  

---

## 🧪 Pruebas

Las pruebas están en:
tests/anchor.test.ts

Se ejecutan directamente desde Solana Playground.

---

## 📦 Despliegue

El despliegue se realiza directamente desde Solana Playground con un solo clic.
